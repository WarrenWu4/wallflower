# Wallflower 

GUI utility application for configuring wallpapers in Hyprland. Built on top of Hyprpaper and Hyprctl

## Installation

### Manual Installation

```bash
git clone https://github.com/WarrenWu4/wallflower
cd wallflower
makepkg -si
```

## Contributing

### Reporting Bugs Or Requesting Features

If you find a bug or would like to request a new feature. Please first open an issue to discuss it and attach the appropriate tag.

### Development Process

1. Fork the project.

2. Create your feature branch. Please following the convention of **feat/issue-#**.

3. Commit your changes. "git commit -m feat: description... "

4. Push to remote branch. "git push origin feat/issue-#"

5. Open a pull request for review.
